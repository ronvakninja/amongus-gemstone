# Amongus Gemstones #
Hypixel skyblock 1.8.9 texture pack for amongus gemstones.  
currently include:  
- Amber
- Amethyst
- Aquamarine
- Citine
- Jade
- Jasper
- Onyx
- Opal
- Peridot
- Ruby
- Sapphire
- Topaz
